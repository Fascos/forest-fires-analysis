# Forest Fire Analysis

## Project Overview

This project analyzes the Forest Fires dataset to identify the factors that influence burned area using multiple linear regression techniques. The analysis follows a structured data science workflow, from data exploration and preprocessing through model development, evaluation, and interpretation.

## Project Objectives

* Explore the Forest Fires dataset.
* Identify the factors associated with burned area.
* Build and compare multiple linear regression models.
* Evaluate model performance using statistical and predictive metrics.
* Draw conclusions and provide recommendations based on the findings.

## Project Workflow

The Jupyter Notebook contains the complete analysis organized into the following sections:

1. Problem Definition and Objectives
2. Importing Required Libraries
3. Loading the Dataset
4. Data Cleaning and Preprocessing
5. Exploratory Data Analysis (EDA)
6. Feature Engineering
   * Interaction Terms
   * Indicator (Dummy) Variables
   * Variable Transformations
7. Model Development

   * Baseline Multiple Linear Regression Model
   * Nonlinear (Quadratic) Model
   * Interaction Model
   * Indicator Variable Model
   * Log-Transformed Model
8. Model Diagnostics and Evaluation

   * Assumption Checking
   * Residual Analysis
   * Model Comparison
   * Performance Metrics
9. Conclusions and Recommendations

## How to Run the Project

1. Clone this repository.
2. Install the required Python libraries.
3. Open `Forest_Fires.ipynb` using Jupyter Notebook or Visual Studio Code.
4. Run the notebook from the first cell to the last cell.
